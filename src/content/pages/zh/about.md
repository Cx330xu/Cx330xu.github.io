我是 **Cx330xu**，AI Agent 研发工程师，长期关注 **LLM 应用、RAG 检索增强与 Agent 工程化落地**。我习惯用可运行的代码验证想法，再用文章把架构决策与踩坑过程写清楚，方便复现与讨论。

## 代表项目

### [Prompt System](/zh/projects/prompt-system/)

基于 FastAPI + Next.js 的提示词管理系统，支持 Linux 本地与 Docker 部署。解决提示词版本分散、难以复用的问题，可作为 Agent / RAG 项目的提示词基础设施。技术关键词：FastAPI、Next.js、Docker、Prompt Engineering。

### [PDF Vector Extractor](/zh/projects/pdf-vector-extractor/)

面向 PDF 的矢量与文本提取工具，服务于后续 RAG 索引与文档处理流水线。技术关键词：JavaScript、Web、文档解析。

更多项目见 [项目列表](/zh/projects/)。
