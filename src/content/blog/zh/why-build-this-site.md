作为一名计算机行业从业者，我希望有一个**长期可维护**的个人站点：

- 展示项目与开源实践
- 沉淀技术文章与短笔记
- 保护隐私，只公开技术能力

本站基于 **Astro** 构建，推送到 GitHub 后自动部署到 `Cx330xu.github.io`。

## 下一步

- 持续更新 Notes 短笔记
- 补充更多项目与深度文章
- 按需启用 Giscus 评论互动

```bash
npm run dev    # 本地预览
git push       # 自动部署
```
